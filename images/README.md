# Profile Image

Place your profile photo here as `profile.png`

## Requirements:
- **Filename**: profile.png (exactly)
- **Recommended size**: 400x400 pixels (will be displayed as 200x200)
- **Format**: PNG with transparent background (preferred) or JPG
- **Aspect ratio**: Square (1:1)

## Tips:
- Use a professional headshot
- Ensure good lighting and clear background
- High resolution is better (it will be automatically scaled down)
- If using JPG, make sure the background is clean and professional

## Alternative:
If you prefer a different format or filename, update the image path in `index.html`:

```html
<img src="images/profile.png" alt="Shreya Kapoor" class="profile-img">
```

Change `images/profile.png` to your preferred path.
